CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation and layout of web pages.

Internal CSS Class Selector

Internal CSS are used to style the HTML element within same HTMl page. Inside the header section we declare our css property.It is declared using # symbol.

Internal CSS ID Selector

ID selector is used to style the HTML element with unique id. It is declared using # symbol.

External CSS

External CSS are used to style the HTML element within different HTML page. It is declared using link tag.

CSS Box Model

The CSS box model is a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

CSS Text

The CSS text properties allow you to set the color, size, and spacing of text.

CSS Colors

CSS allows you to set text and background colors using various color formats.

CSS Borders

Borders can be customized using width, style, color, and radius for rounded edges.
